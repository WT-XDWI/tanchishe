# 🐍 贪吃蛇 · 兼容 IE6

> 一个在 Internet Explorer 6 及所有现代浏览器上均可流畅运行的贪吃蛇游戏，支持键盘、WASD、触摸滑动、屏幕按钮和游戏手柄操控。

---

## ✨ 特性

- **极致兼容**：支持 IE6+、Chrome、Firefox、Edge、Safari 等所有主流浏览器
- **多种操控方式**：
  - ⌨️ **键盘**：方向键 ↑↓←→
  - ⌨️ **WASD**：W/A/S/D
  - 🖱️ **屏幕按钮**：页面下方的方向按钮
  - 👆 **触摸滑动**：在游戏区域直接滑动（移动端）
  - 🎮 **游戏手柄**：连接手柄后，使用左摇杆或方向键（D-Pad）控制
- **智能食物生成**：使用 BFS（广度优先搜索）算法确保生成的每个食物都能被蛇头到达，避免卡死局面
- **最高分记录**：通过 Cookie 在本地持久化保存最高分
- **视觉辅助**：半透明网格线帮助定位，提升操作精度
- **全 DOM 渲染**：不依赖 Canvas，确保 IE6 完全兼容

---

## 🎮 游戏规则

- 控制蛇头移动，吃掉红色食物 🍎
- 每吃一个食物，蛇身长度 +1，得分 +1
- 撞墙或撞到自己身体，游戏结束
- 当蛇身填满整个棋盘（共 20×20 = 400 格）时，您将获胜 🏆

## 💻 系统要求

本游戏设计为极致轻量，几乎可在任何能运行网页浏览器的设备上流畅运行。

- **操作系统**：Windows 95/98/2000/XP/Vista/7/8/10/11、macOS、Linux、Android、iOS 等（只要支持浏览器即可）
- **浏览器**：
  - **最低支持**：Internet Explorer 6 及以上版本（需允许执行脚本）
  - **推荐**：Microsoft Edge、Google Chrome、Mozilla Firefox、Apple Safari、Opera 等现代浏览器（可获得更佳性能与触控体验）
- **屏幕分辨率**：建议 800×600 或更高（游戏区域为 400×400 像素）
- **内存**：至少 16 MB RAM（实际占用远低于此）
- **其他**：需启用 JavaScript（IE6 默认启用，但若提示“限制脚本”，请点击“允许阻止的内容”）

> **注**：所有现代浏览器均能完美运行，IE6 用户请允许脚本执行即可正常游戏。

## 🛠️ 技术实现

- **语言**：纯 JavaScript（ECMAScript 3 语法，兼容 IE6）
- **渲染方式**：DOM 元素操作（`innerHTML`），不使用 Canvas
- **算法**：BFS 用于食物可达性检测，确保生成的每个食物均可到达
- **事件处理**：
  - 键盘事件：`document.onkeydown`（兼容 IE6）
  - 触摸事件：`touchstart` / `touchmove` / `touchend`
  - 手柄事件：`Gamepad API`（仅现代浏览器）
- **数据持久化**：Cookie 存储最高分
